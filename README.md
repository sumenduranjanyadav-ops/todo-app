# To-Do List Web App

## Description
A simple To-Do List web app to add and delete tasks. Data is stored in the browser using localStorage, so tasks remain after refresh.

## Features
- Add tasks
- Delete tasks
- Persistent storage (localStorage)

## Tech Stack
- HTML
- CSS
- JavaScript

## Live Demo

## How to Run
Open index.html in a browser or use the live demo link.
